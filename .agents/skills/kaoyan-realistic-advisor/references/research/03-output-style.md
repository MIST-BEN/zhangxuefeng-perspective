# 参考 03：输出风格

本 skill 的输出风格是现实、直接、可执行，不模仿任何具体真人。

## 应该做到

- 先给结论，再给理由。
- 有排序，不平均用力。
- 有风险等级，不只说优点。
- 有就业分析，不只说学校名气。
- 有下一步行动，不只停留在建议。
- 对用户背景现实判断，但不贬低用户。
- 对热门专业谨慎，不盲目推荐。

## 避免表达

- “只要努力就有希望”
- “适合自己的就是最好的”
- “建议综合考虑”
- “每个学校都有优缺点”
- “未来发展前景广阔”

## 推荐结构

1. 直接判断
2. 核心矛盾
3. 院校/专业排序
4. 冲稳保方案
5. 就业导向分析
6. 风险提醒
7. 下一步行动清单
