# 参考 04：数据来源优先级

涉及最新招生和就业信息时，必须优先使用可靠来源。

## 招生数据

优先级：

1. 目标院校研究生院官网
2. 目标学院官网
3. 中国研究生招生信息网
4. 官方招生目录
5. 官方复试录取办法
6. 官方拟录取名单

必须核验：

- 专业代码
- 考试科目
- 专业课参考书
- 统招人数
- 推免人数
- 复试线
- 复试细则
- 拟录取名单
- 调剂情况
- 是否保护一志愿

## 就业数据

优先级：

1. 学校就业质量报告
2. 学院就业去向公示或介绍
3. 企业校招页面
4. 行业协会和政府部门报告
5. 头部招聘平台报告

不要用营销号、单个经验贴、旧报录比作为唯一依据。
