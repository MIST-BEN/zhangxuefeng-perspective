# 示例对话

## 用户

我是一名普通本科工科学生，本科为某传统工科专业，想报考控制工程专硕，希望去一线城市或目标城市发展，也想兼顾就业和上岸概率。请帮我设计冲稳保方案。

## Skill 输出方向

### 一、直接判断

你的目标可以保留学校层级上的冲刺，但不能只盯着高热度院校。普通本科工科背景、跨到控制工程、目标城市较热门、又希望提高上岸概率，这几个条件叠加后，核心矛盾不是“能不能冲”，而是“冲刺收益和失败成本是否匹配”。

更合理的策略是：高层级院校作为冲刺，目标城市或周边就业不错的工科院校作为稳妥，再准备一个你能接受的保底。试错成本较低的人，不能把一志愿当情绪表达。

### 二、核心矛盾

你同时想要：

- 目标城市资源
- 学校层级提升
- 控制工程方向
- 就业结果不错
- 上岸概率可控

这些目标都合理，但放在一起难度会明显上升。本科背景和复习基础会带来复试不确定性，所以初试必须尽量打高，不能指望复试逆袭。

### 三、院校/专业排序

| 推荐层级 | 院校类型 | 判断 |
|---|---|---|
| 第一优先 | 目标城市或周边控制/自动化资源较强、统招人数较稳的院校 | 主攻档，兼顾城市和录取概率 |
| 可以冲 | 层级更高、就业资源更强的控制工程相关方向 | 有收益，但风险不低 |
| 可以稳 | 工科基础较好、复试规则清晰、专业方向匹配的院校 | 更符合上岸优先 |
| 可以保 | 专业就业尚可、复试风险较低、统招人数充足的学校 | 保底也要有就业下限 |
| 不建议 | 顶级热门院校或统招极少的热门专业 | 和风险控制目标冲突 |

### 四、冲稳保方案

| 档位 | 学校类型 | 推荐理由 | 最大风险 | 适合程度 |
|---|---|---|---|---|
| 冲 | 高层级工科院校控制工程/自动化相关方向 | 学校层级和行业资源有简历增益 | 复试竞争、本科背景、专业课变化 | 适合数学和专业课能明显拉分时 |
| 稳 | 目标城市或周边工科强校控制工程方向 | 更符合上岸优先，专业就业不差 | 学校层级可能略有牺牲 | 适合作为主推档 |
| 保 | 统招人数较足、复试规则清晰、专业方向能接受的学校 | 降低一志愿失败概率 | 城市或层级可能牺牲 | 适合试错成本较低的底线方案 |

### 五、就业导向分析

控制工程可以往工业控制、智能制造、轨道交通、嵌入式系统、自动化设备等方向靠。它适合传统工科背景学生转入，但不能只靠研究生专业名称获得高薪岗位。

必须补：

- 自动控制原理
- 现代控制基础
- C/C++ 或 Python
- 单片机/嵌入式基础
- Matlab/Simulink
- 一个可讲清楚的控制工程相关项目

### 六、风险提醒

- 复试线不等于真实录取难度。
- 拟录取名单比分数线更重要。
- 统招人数少的专业风险会被放大。
- 专业课换科目会导致备考风险上升。
- 是否保护一志愿要看复试细则和拟录取名单。
- 热门城市、热门学校层级、热门工科方向叠加，不能当低风险目标。

### 七、下一步行动清单

1. 列出10所候选学校。
2. 查每所学校最新招生目录。
3. 查近3年拟录取名单。
4. 统计初试最低分、中位分和高分段。
5. 记录统招人数和推免人数。
6. 查专业课参考书、真题和是否改科目。
7. 查复试科目、复试占比、是否有机试/项目问答。
8. 给每所学校标风险等级。
9. 在正式报名前确定最终一志愿。
10. 同步准备一个可用于复试的控制工程相关项目。
